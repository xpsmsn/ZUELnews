# 章鱼的高效助手小程序

## 项目概述

章鱼的高效助手是一个微信小程序新闻获取工具，面向校园新闻的抓取、清洗与阅读体验优化。

## 主要功能

### 1. 校园新闻提取

* 支持从中南财经政法大学文澜新闻、本科生院与校团委网站自动提取新闻

* 提供最近新闻列表展示与详情阅读

### 2. 网页浏览

* 在小程序内直接打开新闻链接

* 支持复制链接到浏览器打开

## 项目结构

```
├── pages/
│   ├── index/          # 首页，新闻列表展示
│   ├── news-detail/    # 新闻详情页
│   └── web-view/       # 网页浏览页
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── project.config.json # 项目配置
└── sitemap.json        # 站点地图
```

## 技术实现

### 前端

* 微信小程序原生开发

* 使用 `web-view` 组件实现网页浏览

## 使用说明

### 1. 新闻浏览

1. 打开小程序首页
2. 选择要查看的网站（文澜新闻、本科生院、校团委）
3. 系统会自动提取并展示新闻
4. 点击新闻标题查看详情

## 开发说明

### 环境要求

* 微信开发者工具

* 微信小程序账号

### 运行方式

1. 在微信开发者工具中导入项目
2. 填写小程序 AppID
3. 运行小程序
