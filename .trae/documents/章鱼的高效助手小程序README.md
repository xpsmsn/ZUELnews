# 章鱼的高效助手小程序

## 项目概述

章鱼的高效助手是一个功能强大的微信小程序，旨在帮助用户快速获取和处理校园新闻信息，集成了新闻提取、AI内容处理和知识库查询等功能。

## 主要功能

### 1. 校园新闻提取

* 支持从中南财经政法大学本科生院和校团委网站自动提取新闻

* 智能过滤近7天的最新新闻

* 详细的新闻内容展示

### 2. AI内容处理

* 支持多种AI提供商：硅基流动、OpenAI、DeepSeek

* 自动提取新闻正文并生成简洁摘要

* 可自定义API配置

### 3. Dify知识库查询

* 集成Dify知识库，支持智能问答

* 支持会话管理，保持上下文对话

### 4. 网页浏览

* 在小程序内直接打开新闻链接

* 支持复制链接到浏览器打开

## 项目结构

```
├── pages/
│   ├── index/          # 首页，新闻列表展示
│   ├── news/           # 新闻详情页
│   ├── ai-process/     # AI内容处理页
│   └── web-view/       # 网页浏览页
├── app.js              # 小程序入口
├── app.json            # 小程序配置
├── app.wxss            # 全局样式
├── jwc_news_extractor.py  # 本科生院新闻提取脚本

├── project.config.json     # 项目配置
└── sitemap.json            # 站点地图
```

## 使用说明

### 1. 新闻浏览

1. 打开小程序首页
2. 选择要查看的网站（本科生院或校团委）
3. 系统会自动提取并展示近7天的新闻
4. 点击新闻标题查看详情

### 2. AI内容处理

1. 在新闻详情页点击"AI处理"
2. 系统会自动提取新闻正文
3. 选择AI处理方式（目前支持摘要）
4. 点击"开始处理"按钮
5. 查看AI生成的结果

### 3. Dify知识库查询

1. 在AI处理页面下方的Dify查询区域
2. 输入查询问题
3. 点击"查询知识库"按钮
4. 查看查询结果

### 4. API配置

1. 在AI处理页面点击"设置"按钮
2. 配置AI提供商、API密钥和模型
3. 配置Dify API密钥
4. 点击"保存设置"按钮

## 注意事项

1. **API密钥配置**：使用AI功能需要配置相应的API密钥，请确保密钥的正确性
2. **网络连接**：新闻提取和AI处理需要良好的网络连接
3. **响应时间**：AI处理可能需要一定时间，请耐心等待
4. **内容准确性**：AI生成的内容仅供参考，可能存在一定误差

## 开发说明

### 环境要求

* 微信开发者工具

* Python 3.6+

* 微信小程序账号

### 运行方式

1. 在微信开发者工具中导入项目
2. 填写小程序AppID
3. 运行小程序

### 新闻提取脚本

Python脚本用于辅助开发和测试，可单独运行：

```bash
python jwc_news_extractor.py
python tw_news_extractor.py
```
